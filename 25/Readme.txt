В файлах EEH - данные по энергии, энтропии и Хёрсту с окном в 30 секунд, в файлах EEH2 - те же данные, но по периодам.
В файлах INFO добавились следующие столбцы: 

TP - сколько очков есть в запасе игрока (этап Contribution)

LP - сколько очков есть в запасе у команды (этап Extract)

SUM - сколько очков набрала команда игрока к данному моменту времени (оба этапа)

TH - цель команды (этап Contribution)
